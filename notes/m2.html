  • Submatrix

  • Row Echelon Form
    • Zero rows at bottom
    • Pivot entry (first non-zero entry) in any row is 1
    • Pivot entry column is right of pivot entry column of prev. row
  • Reduced REF
    • All entries above pivot entries are zero
  • Inverse

  • T(A) B <=> T(A B)

    Where T is a transformation

  • T(A + B) <=> T(A) + T(B)

    Where T is a transformation

  • Rank of Matrix

    No. of nonzero rows in REF of Matrix

  • Homogenous system

    AX = B and B = 0

  • Non-homogenous system

    AX = B and B != 0

  • Soln of NH system
    • Unique

      rank(A) = rank([A:B]) = n

    • Infinitely many

  • Soln of H system

    • Infinitely many

  • Linear Transformation

  • Every Matrix Transformation is Linear
  • How to find matrix for T

    T(Yi) = Zi, matrix A => [Z0 Z1 …]
    s.t. Yi = [1, 0, 0 …] (column)
    with 1 at the ith position
    A is called Standard Matrix of T

  • Vector Space

    Set V of elements called vectors
    has operations of addition and scalar mult.

    • Closure Axioms
      1. The sum u + v exists and b.t. V
      2. cu b.t. V
    • Addition Axioms
      1. u + v = v + u (commutative)
      2. u + (v + w) = (u + v) + w (associative)
      3. t.e. 0 s.t. u + 0 = u
      4. f.a. u b.t. V t.e. -u s.t. u + (-u) = 0
    • Scalar Mult. Axioms
      1. c(u + v) = cu + cv
      2. (c + d)u = cu + du
      3. c(du) = (cd)u
      4. 1u = u
  • Pointwise Addition

    (f + g)(x) = f(x) + g(x)

  • Vector Subspace

    A subset U of V s.t.
    U is closed under addition and scalar mult.

    • U contains the zero vector of V

      bc 0u = Z s.t. Z = zero vect

  • Linear Combination

    W is linear combination of v1, …, vn
    if W = c1v1 + … + cnvn
    for some scalars c1 … cn

  • Spanning vectors

    Let v1…vn b.t. vector space V
    they span V if every vector of V can be
    expressed as a linear combination of them

  • Linear Dependence

    Given set of vectors {v1 .. vn} in V

    • Linearly Dependent

      t.e. {c1 .. cn} s.t. c1..cn not all zero
      and c1v1 + … + cnvn = 0

    • Linearly Independent

      c1v1 + … + cnvn = 0 only for c1,..cn all 0

  • Vect A and B are lin. dep

    iff B is scalar mult. of A

  • 3 vects lin. indep. iff not lie in plane
  • if set contains zero vect, lin dep.
  • if subset of A is lin dep, A is lin dep
  • Basis of Vector Space

    Set {v1 .. vn} is basis of v.s. V
    if it spans V and is lin. indep.

  • if basis has n vects

    any set > n vects big is lin dep.

    • hence, all basis have some no. of vects
  • Rank of matrix
    • no. of lin. indep. rows or columns
    • nonzero rows in RREF
  • Basis

    Non-zero rows of RREF of spanning vector matrix

  • Eigenvalue & Eigenvector

    Let A be nxn matrix
    Ax = bx
    Scalar b => eigenvalue
    Vector x => eigenvector corr. to b

  • Finding Eigenvalue

    mod(A - bI) = 0
    s.t. A = matrix and b = eigenval

    • Eigenvector is found by

      (A - bI)x = 0

  • Eigenspace

    Set of all eigenvectors for a given eigenvalue

  • Kernel

    wrt linear transformation T,
    ker(T) = Set of vectors mapped to zero vector
    aka null space, null(T)

  • Range

    wrt linear transformation T,
    range(T) = Set of vectors that are images
    of vectors in domain

  • Rank-Nullity Theorem

    rank(T) + nullity(T) = dim(domain(T))
    s.t. nullity = dim(ker(T))

  • Range and Rank

    dim(range(T)) = rank(T)

  • Fundamental Spaces

    orth(row(A)) = null(A)
    orth(col(A)) = null(trans(A))

  • One-to-one Transformation

    T(a) = T(b) => a = b
    ker(T) = {0}
    If T is linear, T is an isomorphism.

  • Onto Transformation

    T : U -> V
    if f.a. y b.t. V, t.e. x b.t. U
    s.t. T(x) = y, then T is Onto

  • Nonsingular Transformation

    T(x) = Ax s.t. |A| != 0

  • Invertible Transformation

    All of the following statements are equivalent.

    1. T is invertible
    2. T is nonsingular (|A|!=0)
    3. T is one-to-one
    4. ker(T) is the zero vect
    5. T is onto the domain
    6. inv(T) is linear
    7. inv(T) is defined by inv(A)
  • Coordinate vector

    Given basis = {u1 … un}
    v = a1u1 + … + anun
    Coordinate vector vb is given by
    column vector [a1, …, an]

  • Change of bases

    T : A -> A'
    Let coord vect x be in basis A
    Then T(x) = x in basis A'

  • Similarity

    Matrix A' is similar to matrix A
    if t.e. inv.ble matrix C
    s.t. A' = inv(C) A C
